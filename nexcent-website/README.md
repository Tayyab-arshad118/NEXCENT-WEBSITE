# Nexcent Studios Website

A modern, responsive static website for Nexcent Studios, a software house offering services and internship programs.

## Features
- Modern, minimal design with a dark theme and green accent colors
- Responsive layout for all devices
- Animated navigation bar and footer on all pages
- Homepage with hero section, services overview, internship teaser, and CTAs
- Services page with detailed offerings
- Internship page with disabled registration form and "Coming Soon" tooltip
- Courses page as a placeholder
- About page with company story and founders
- Contact page with form and social links
- Smooth scroll and hover animations
- Clean, modern fonts (Poppins, Inter)

## Setup & Usage

1. Clone or download this repository.
2. Open `index.html` or any page in your browser.
3. All assets are local; no build step required.

## Customization
- Place your logo and images in `assets/images/`.
- Add custom fonts to `assets/fonts/` or use Google Fonts in `styles.css`.
- Update contact/social info in `contact.html` and footer.

## ğŸš€ Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/nexcent-website.git
```

2. Navigate to the project directory:
```bash
cd nexcent-website
```

3. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```

4. Visit `http://localhost:8000` in your browser

## ğŸ’» Usage

### Development

The website uses vanilla HTML, CSS, and JavaScript, so no build process is required. Simply edit the files and refresh your browser to see changes.

### Customization

- **Colors:** Modify CSS variables in `assets/css/styles.css`
- **Content:** Update HTML files directly
- **Forms:** Configure form handlers in JavaScript files

### Deployment

The site is configured for Netlify deployment. Simply connect your repository to Netlify for automatic deployments on push.
